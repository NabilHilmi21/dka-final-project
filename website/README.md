# Streamlit Website

Website ini menampilkan alur fuzzy logic risiko kecelakaan dari awal sampai akhir:

1. Membaca dataset
2. Preprocessing dan data cleaning
3. Input/output fuzzy
4. Fungsi keanggotaan
5. Fuzzifikasi
6. Rule base
7. Inferensi
8. Defuzzifikasi Mamdani dan Sugeno
9. Hasil semua data
10. Evaluasi akurasi, MAE, dan confusion matrix
11. Visualisasi perbandingan
12. Interpretasi hasil

## Cara Menjalankan

Jalankan dari folder root project:

```bash
streamlit run website/main.py
```

Jika dependency belum ada:

```bash
pip install -r website/requirements.txt
```
